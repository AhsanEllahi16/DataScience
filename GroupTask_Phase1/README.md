# GroupTask_Phase1

## Overview
Phase 1 submission: Import, clean, and visualize inflation and interest rate data from World Bank (WDI).

## Folder Structure
- **data/**: Contains raw and cleaned CSV files
- **scripts/**: Contains R script used for analysis
- **outputs/**: Contains generated plots
- **docs/**: Contains report and documentation

## How to Run
1. Open `scripts/IDS_GroupTask_phase1.R` in RStudio.
2. Run all lines to reproduce results.
3. Output plots and CSVs will appear in `outputs/` and `data/` folders.
